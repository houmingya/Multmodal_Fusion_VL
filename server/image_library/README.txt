# ====================================
# 图片库说明文件
# ====================================

## 📁 图片库目录用途

此目录用于存放「文搜图」功能的检索图片库。

## 🖼️ 支持的图片格式

- JPG / JPEG
- PNG
- BMP
- WEBP

## 📝 使用说明

### 1. 添加图片
将待检索的图片直接复制到此目录，例如：
- cat.jpg（猫的照片）
- dog.jpg（狗的照片）
- sunset.jpg（日落风景）
- city_night.png（城市夜景）

### 2. 命名建议
- 使用有意义的文件名（便于识别）
- 避免中文文件名（防止编码问题）
- 使用小写字母和下划线

### 3. 图片数量
- 最少：1张（用于测试）
- 推荐：10-100张（实用场景）
- 最大：无限制（但会影响检索速度）

### 4. 图片质量
- 分辨率：建议 ≥ 224x224 像素
- 大小：每张 < 10MB
- 格式：清晰、无水印

## 🔄 更新图片库

### 方法1：手动更新
1. 在此目录添加/删除图片
2. 重启服务器（会自动重建索引）

### 方法2：动态更新（未来功能）
通过API接口上传图片，无需重启服务器。

## 🧪 测试图片下载

### 方法1：使用自己的图片
从本地相册选择图片复制到此目录。

### 方法2：从网上下载示例图片
```bash
# Linux/Mac
cd image_library
wget https://example.com/cat.jpg
wget https://example.com/dog.jpg

# Windows PowerShell
cd image_library
Invoke-WebRequest -Uri "https://example.com/cat.jpg" -OutFile "cat.jpg"
```

### 方法3：使用Unsplash免费图片
访问 https://unsplash.com/ 下载高质量免费图片。

## ⚠️ 注意事项

1. **版权问题**：确保图片有使用权限
2. **隐私保护**：不要上传敏感个人照片到公网服务器
3. **定期清理**：删除不需要的图片释放空间
